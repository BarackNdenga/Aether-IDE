# Changelog

## [1.0.0] - 2025-01-01

### Added
- Save workspace state (open editors, cursors, scroll positions)
- Restore latest saved state
- List and select from last 5 saves
- Clear all saved states
- Keyboard shortcuts `Ctrl+Alt+S` / `Ctrl+Alt+R`
- Multi-column editor support
- Missing file handling on restore
